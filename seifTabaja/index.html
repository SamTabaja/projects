Push Pin Map

I looooove travelling and I want to travel all over the world. I always wanted to buy the Push Pin Travel Map but I could not find it. Therefore, I decided to create my own but this time as soft map. Using Google`s maps API, I came up with Push Pin Map which allows me to Pin every city I travel to. Also, it allows me to find the coordinates of any place I want on the planet by only one click on the map and Pin too. I made it responsive so that I can use it on my computer and also on my super smart phone.

Word Sprint

Word Sprint is nice and challenging game. It is totally about javascipt. In this game, you have to type a givin word within a givin time. The player can choose the time needed to write the givin word which indicates the difficulty level. Every time the player correctly type the word, a point will be added to the score and the progress bar will be incremeneted. If the time outs, the game overs.
